# Hướng dẫn nhanh: làm website 1 trang & SEO cơ bản

## 1) Cấu trúc tối thiểu cho người mới
- **Bạn cần 2 file**: `index.html` (nội dung) + `styles.css` (giao diện).
- Khi muốn xuất bản, chỉ cần upload 2 file này lên hosting.

## 2) Các bước làm website từ con số 0
1. **Chuẩn bị nội dung**: viết sẵn mục tiêu, danh mục sản phẩm, set ngân sách, quy trình, liên hệ.
2. **Tạo file HTML**:
   - Thêm tiêu đề, mô tả, hình ảnh (nếu có) và các section.
   - Chèn link file CSS.
3. **Tạo file CSS**:
   - Định màu thương hiệu, font chữ, layout.
   - Làm responsive cho mobile.
4. **Xem thử trên máy**: mở `index.html` bằng trình duyệt để kiểm tra.
5. **Xuất bản**:
   - Cách dễ nhất: dùng hosting tĩnh (GitHub Pages, Netlify) hoặc thuê hosting truyền thống rồi upload.
6. **Cập nhật nội dung định kỳ**: thêm hình ảnh thực tế, case study, phản hồi khách hàng.

## 3) SEO cơ bản cho website 1 trang
- **Title & Description rõ ràng**: mô tả đúng dịch vụ quà tặng doanh nghiệp.
- **Heading hợp lý**: 1 thẻ `<h1>` duy nhất, các mục quan trọng dùng `<h2>`.
- **Từ khóa tự nhiên**: lặp vừa đủ các cụm như “quà tặng doanh nghiệp”, “quà tặng khách hàng”.
- **Tốc độ tải nhanh**: hạn chế ảnh quá nặng, dùng ảnh nén.
- **Liên hệ rõ ràng**: đặt hotline, email, địa chỉ để tăng độ tin cậy.
- **Schema & Google Business**: nếu có địa điểm kinh doanh, đăng Google Business Profile.
- **Nội dung cập nhật**: đăng bài tư vấn, case study, hình ảnh dự án để tăng uy tín.

## 4) Gợi ý để tăng hiệu quả chuyển đổi
- Thêm nút gọi nhanh (hotline) và nút Zalo/Email.
- Hiển thị hình ảnh thực tế của bộ quà tặng.
- Đưa bảng giá tham khảo và cam kết số lượng tối thiểu.
