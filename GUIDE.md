# Hướng dẫn nhanh: làm website 1 trang & SEO cơ bản

## 1) Bạn cần chuẩn bị gì?
- **Nội dung**: danh mục sản phẩm, set ngân sách, quy trình, thông tin liên hệ.
- **Logo & màu thương hiệu** (nếu có) để đồng bộ nhận diện.
- **2 file**: `index.html` (nội dung) + `styles.css` (giao diện).

## 2) Các bước làm website từ con số 0
1. **Viết nội dung** theo từng mục bạn muốn hiển thị.
2. **Tạo file HTML**:
   - Thêm tiêu đề (`<title>`) và mô tả (`<meta name="description">`).
   - Dùng các section rõ ràng: Danh mục, Set ngân sách, Theo yêu cầu, Liên hệ.
3. **Tạo file CSS**:
   - Đặt màu thương hiệu, font chữ, khoảng cách.
   - Làm responsive cho mobile.
4. **Xem thử trên máy**: mở `index.html` bằng trình duyệt.
5. **Xuất bản**:
   - Dễ nhất: dùng hosting tĩnh (GitHub Pages, Netlify).
   - Hoặc thuê hosting truyền thống rồi upload 2 file.
6. **Cập nhật định kỳ**: thêm hình ảnh thực tế, dự án đã làm, phản hồi khách hàng.

## 3) SEO cơ bản cho website 1 trang
- **Title & Description rõ ràng**: mô tả đúng dịch vụ quà tặng doanh nghiệp.
- **Heading hợp lý**: 1 thẻ `<h1>` duy nhất, các mục quan trọng dùng `<h2>`.
- **Từ khóa tự nhiên**: “quà tặng doanh nghiệp”, “quà tặng khách hàng”, “in logo”.
- **Tốc độ tải nhanh**: hạn chế ảnh quá nặng, dùng ảnh nén.
- **Liên hệ rõ ràng**: đặt hotline, email, địa chỉ để tăng độ tin cậy.
- **Google Business Profile**: nếu có địa điểm, nên đăng ký để tăng hiển thị địa phương.

## 4) Mẹo tăng hiệu quả chuyển đổi
- Thêm nút gọi nhanh (hotline) và nút Zalo/Email.
- Hiển thị ảnh thật của bộ quà tặng & đóng gói.
- Thêm bảng giá tham khảo và cam kết số lượng tối thiểu.
- Đặt form liên hệ ngắn gọn để khách gửi yêu cầu nhanh.
